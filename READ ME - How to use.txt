ORGANIZE A FOLDER WITH AI — how to use
=======================================

What it does: you pick one of your folders (with decks, spreadsheets, PDFs,
notes) and the AI sets it up so it can navigate that folder and help you — and
it keeps itself updated as you add files.


STEP 1 — Put the right file on your Desktop
-------------------------------------------
  • On Mac:      "Organize a folder with AI.command"
  • On Windows:  "Organize a folder with AI.bat"

(Use only the one for your computer. Drag it to your Desktop.)


STEP 2 — Have an assistant installed (one time only)
----------------------------------------------------
You need two things, once per computer. If either is missing, the app opens the
install link for you. Ask a technical person to help the first time if needed.

  1) An assistant — Claude Code OR Codex:
       • Claude Code:  https://claude.com/claude-code
       • Codex:        https://learn.chatgpt.com/docs/codex/cli

  2) Python — powers the automatic "what changed" tracking:
       • https://www.python.org/downloads/
     (You can start without it — setup and chat still work — but the automatic
      tracking only turns on once Python is installed.)


STEP 3 — Use it
---------------
1) Double-click the file.

   >> The FIRST time, your computer shows a safety warning (normal, because the
      file came from outside):
      • Mac:      right-click the file -> Open -> Open
      • Windows:  "More info" -> "Run anyway"

2) Choose the folder you want to organize (a little window opens).

3) Choose the assistant: 1 = Claude Code, 2 = Codex.

4) Done — the AI starts working right in front of you. It may ask you a few
   questions about the folder (what it is, what to skip). That's normal and helps
   it get things right.


AFTER THAT
----------
Whenever you want to chat with that folder's files, double-click the "Talk to my
files" file that appears inside it. And when you create new folders, just open the
chat inside them and the AI takes care of organizing.


Questions? Ask whoever sent you this file.
